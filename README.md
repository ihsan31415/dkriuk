# DKriuk Project

This project has been restructured into a Flask backend and a React frontend.

## Prerequisites

- Python 3.x
- Node.js and npm

## Setup and Run

### Backend (Flask)

1. Navigate to the `backend` directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the server:
   ```bash
   python app.py
   ```
   The backend will run on `http://localhost:5000`.

### Frontend (React)

1. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
   The frontend will run on `http://localhost:3000`.

## Features

- **Admin Dashboard**: Monitor stock, revenue, and critical alerts.
- **Distribusi Stok**: Manage stock distribution from the central hub to outlets.
- **Laporan & Analitik**: View sales reports and charts.
- **POS Outlet**: Point of Sale system for outlets.
